## What to Do

1. **Open the documentation link above**
2. **Look for "Classes" or "API Reference" section**
3. **Find the main entry point class** (could be named differently than we expected)
4. **Check for:**
   - Initialization methods
   - Tool registration methods
   - Query/request processing methods

## Common Patterns in Apple Frameworks

Apple frameworks often use:
- `Model` suffix (e.g., `LanguageModel`, `FoundationModel`)
- `Session` suffix (e.g., `ModelSession`, `LLMSession`)
- `Request` suffix (e.g., `ModelRequest`, `LLMRequest`)
- `Manager` suffix (e.g., `ModelManager`)
